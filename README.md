# AssetVerse
AssetVerse is a role based website where user's can manage their assets easily like adding assets, deleting etc.Where an Employee can be affiliated through a request and the HR can handle mutiple requests from other side.In this site HR can upgrade his employee affiliation number also as it is limited.Users can observe their request by various status.Everything is seamless and smooth.
Live-link:https://asset-management-b021d.web.app/
## Features
* Easy to manage assets.
* It is secure, fast and responsive for small devices.
* Interactive floor plan
* Framer motion transitions

## Tech Stack
* Frontend: React, Tailwind CSS
* Backend: Node.js, Express
* Database: MongoDB
## How to install 
1. Clone the repository: client: https://github.com/prinsahmed/Asset-client.git server:https://github.com/prinsahmed/Asset-server.git
2. Install dependencies: **npm install**
3. Set up your environment variables **(create a .env file)**
4. Start the app: **npm run dev**
